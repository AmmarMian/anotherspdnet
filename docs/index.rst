Welcome to AnotherSPDNet's documentation!
=========================================

This is a work in progress implementation of SPDNet architecture in pytorch.
While other implementations exist (see 
`Davoudi's implementation <https://github.com/adavoudi/spdnet/tree/master>`_
or `torchspdnet <https://gitlab.lip6.fr/schwander/torchspdnet>`_), this work
aims at rebuilding to better my understanding of pytorch mechanics and will be
a base on which further improve the model.
